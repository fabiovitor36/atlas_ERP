# Atlas ERP

## Visão do Produto

O Atlas ERP é um sistema SaaS desenvolvido para atender distribuidoras de bebidas, conveniências, adegas e pequenos comércios.

Nosso objetivo é fornecer uma plataforma simples, rápida, moderna e inteligente para controlar:

- Estoque
- Compras
- Vendas
- Delivery
- Financeiro
- Clientes
- Fornecedores
- Relatórios
- Dashboard
- Inteligência Artificial

## Missão

Automatizar pequenas empresas com tecnologia de alto nível.

## Público-alvo

- Distribuidoras
- Adegas
- Conveniências
- Mercados

## Modelo de Negócio

Software como Serviço (SaaS).

## Primeiro Cliente

Distribuidora do projeto Atlas.
